Softwares

O que é um software?

Software é toda a parte lógica de um dispositivo. Se o hardware é o corpo (peças que você pode tocar), o software é o "pensamento" ou as instruções que dizem ao corpo o que fazer.

O software é dividido em categorias, dependendo da função que ele exerce:

➔ Software de Sistema (Básicos):
✱ Garante o funcionamento da máquina.
✱ Controla o hardware e os resources.
Exemplos: Windows, Android, drivers e BIOS.
➔ Software de Aplicativo (Utilitários):
✱ Executa tarefas específicas para o usuário.
✱ Atende necessidades de trabalho ou lazer.
Exemplos: WhatsApp, Word, Netflix e navegadores.
➔ Software de Programação:
✱ Ferramentas usadas para criar novos programas.
✱ Traduz código humano para a máquina.
Exemplos: Editores de código e compiladores.

Os softwares de programação (ou ferramentas de desenvolvimento) são programas que os programadores usam para criar, testar, depurar e manter outros softwares. Eles servem como a ponte entre a lógica humana e o código que o computador consegue executar

Principais Categorias e Exemplos
Ambientes de Desenvolvimento Integrado (IDEs):

Aplicativos completos que reúnem editor de texto, ferramentas de automação e depuradores em uma única interface.

Exemplos: Visual Studio Code, IntelliJ IDEA, Eclipse
Compiladores e Interpretadores:

Softwares que traduzem o código escrito por humanos para a linguagem de máquina (código binário) do computador.

Exemplos: GCC (para C/C++), interpretador Python, Java Virtual Machine (JVM).
Depuradores (Debuggers):

Programas que permitem testar o código passo a passo para encontrar e corrigir erros (bugs) de lógica.

Exemplos: GDB, ferramentas de depuração integradas do Chrome DevTools.
Sistemas de Controle de Versão:

Ferramentas que registram o histórico de alterações no código e permitem que várias pessoas trabalhem no mesmo projeto.

Exemplos: Git (gerenciado em plataformas como GitHub ou GitLab).
